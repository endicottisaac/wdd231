Pathway to Fitness Site Plan

This planning document outlines the visual direction, audience goals, and home page structure for a fitness tracker and workout planner website.

Site Name

Pathway to Fitness

The name was selected because it clearly communicates two core ideas: fitness progress and workout planning. It feels specific to the project purpose and is easy for users to remember since we are in BYU Pathways.

Optional domain idea: pathwaytofitness.com

Site Purpose

Pathway to Fitness will provide a simple hub where users can browse an exercise library, review movement details, build basic workouts, and save workout selections for future sessions. The site will present useful exercise information in a clean and responsive format while supporting interaction through filtering, saving, and modal-based details.

Scenarios

  • Which exercises can I do for legs if I do not have gym equipment?
  • How difficult is an exercise before I add it to a workout?
  • Where can I view the exercises I already saved for my next workout?
